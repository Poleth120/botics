Crear Computadora

Se requiere un nombre de host.
El nombre de host debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar el serial del cpu.
El serial del cpu debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar el serial del monitor.
El serial del monitor debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar el serial del teclado.
El serial del teclado debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar el código de bien del cpu.
El código de bien del cpu debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar el código de bien del Monitor.
El código de bien del monitor debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar el código de bien del teclado.
El código de bien del teclado debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar el modelo del equipo.
El modelo del equipo debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar la capacidad de memoria del equipo.
La capacidad de memoria del equipo debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar la capacidad de memoria RAM del equipo.
La capacidad de memoria RAM del equipo debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar el tipo de procesador que tiene el equipo.
El tipo de procesador del equipo debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar el sistema operativo que tiene el equipo.
El sistema operativo del equipo debe tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar los detalles específicos del equipo.
Los detalles específicos del equipo deben tener al menos 3 caracteres y no mas de 30 caracteres.

Debe ingresar cualquier observación sobre el equipo.
Las observaciones del equipo deben tener al menos 3 caracteres y no mas de 30 caracteres.
